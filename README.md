# weather-app-using-openweather-API
A desktop weather application built with Python and PyQt5.

**Features**

Search weather by city name

Displays:

Current, minimum, and maximum temperature

Humidity

Wind speed

Visibility

Cloudiness

Atmospheric pressure

Displays weather icon from API

Handles invalid input and network errors

**Installation**

You will need to pip install PyQt5 requests

**Usage**

Run the application:

weather_app.py

**API Key**

This project uses the OpenWeather API.

Replace the API key in the code:

api_key = "it_should_be_you_api_key_not_mine"
